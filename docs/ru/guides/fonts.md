# Шрифты

kou не включает шрифты в комплект. Вместо этого, подобно [ort](https://crates.io/crates/ort) для ONNX Runtime, она загружает подобранное семейство в общий кеш (однократно, затем кешируется для всех запусков) и прозрачно находит его. [`FontSet`] объединяет основной моноширинный шрифт с опциональным CJK-запасным вариантом; рендерер загружает их с помощью `ab_glyph` и для каждой кодовой точки выбирает первый шрифт, который её покрывает — так что один рендер смешивает латиницу и CJK без символов-заглушек.

## Семейства

| Слот | Значение `KOU_FONT_*` | Семейство |
|------|--------------------|--------|
| основной (по умолчанию) | `fira-code` | Fira Code |
| основной | `jetbrains-mono` | JetBrains Mono |
| CJK (по умолчанию) | `sarasa` | Sarasa Mono SC (更纱黑体) |
| CJK | `sourcehansans` | Source Han Sans SC (思源黑体) |
| CJK | `smileysans` | Smiley Sans (得意黑) |
| CJK | `none` | отключить запасной вариант |

## Порядок разрешения

1. **Явный файл** — `KOU_FONT_PATH` (основной) / `KOU_FONT_CJK_PATH`.
2. **Общий кеш** — `<cache>/kou/fonts/<family>.ttf|otf`.
3. **Загрузка во время выполнения** — функция `font-fetch` загружает семейство из публичного источника (GitHub / jsDelivr) в кеш.

## Настройки

| Переменная | Назначение |
|-----|---------|
| `KOU_FONT_PRIMARY` | Переопределить основное семейство. |
| `KOU_FONT_CJK` | Переопределить / отключить CJK-семейство. |
| `KOU_FONT_MIRROR` | Заменить канонический хост GitHub / jsDelivr на зеркало (например, для использования за ограничительной сетью). |
| `KOU_DOWNLOAD_PROXY` | Маршрутизировать загрузки шрифтов через `http://` / `https://` / `socks5://` прокси. |
| `KOU_DOWNLOAD_TIMEOUT_SECS` | Таймаут на запрос (по умолчанию 120). |
| `KOU_SKIP_FONT_FETCH` | Отключить загрузку во время выполнения (использовать только кеш / явные пути). |

## Без шрифтов вообще

Если ни один шрифт не может быть разрешён, `FontCache` пуст и рендерер переходит в упрощённый режим, рисуя сплошной блок для каждой непустой ячейки — так что kou всё равно выдаёт *что-то* без шрифтов и никогда не паникует при отсутствующем файле.
