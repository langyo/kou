# kou

**Автоматизация виртуального терминала — PTY + настоящий экран VT100 + шрифты в стиле ort + графические протоколы in-band.**

kou — это самостоятельный движок виртуального терминала: управление PTY,
настоящий эмулятор экрана VT100/ANSI и отрисовка экрана, которая действительно
рисует глифы. Это ядро vtty, выделенное из упаковщика tairitsu и оформленное как
самостоятельная библиотека и CLI.

Три особенности: настоящий экран на базе [`vte`](https://crates.io/crates/vte)
(поддержка CSI/SGR), конвейер шрифтов в стиле ort, который по запросу скачивает
Fira Code / Source Han / Sarasa / Smiley Sans (с зеркалом/прокси), и возможность
in-band-графики: кадр описывается совместимому терминалу (kitty / iTerm2) для
попиксельного отображения прямо в потоке.

Полный список возможностей и API см. в корневом [README](../../README.md).

> В разработке; API может измениться в будущем.
