# kou

**أتمتة الطرفية الافتراضية —— PTY + شاشة VT100 حقيقية + خطوط بأسلوب ort + بروتوكولات رسومية داخل النطاق.**

kou محرّك طرفية افتراضية مستقل——إدارة PTY، ومحاكٍ حقيقي لشاشة VT100/ANSI، وعرض للشاشة
يرسم المحارف فعلًا. وهو قلب vtty المستخرج من مُغلِّف tairitsu والمُعزَّز ليكون مكتبة
وأداة سطر أوامر مستقلة.

يميّزه ثلاثة أمور: شاشة حقيقية مدفوعة بـ[`vte`](https://crates.io/crates/vte) (دعم
CSI/SGR)، وخط أنابيب للخطوط بأسلوب ort يجلب Fira Code وSource Han وSarasa وSmiley Sans
عند الطلب (مع دعم المرآة/الوسيط)، وقدرة رسومية داخل النطاق تصف الإطار للطرفيات المدعومة
(kitty / iTerm2) لعرض البكسلات في موضعها.

لقائمة الميزات كاملةً والـAPI، راجع [README](../../README.md) الجذري.

> لا يزال قيد التطوير؛ قد يتغير الـAPI مستقبلًا.
