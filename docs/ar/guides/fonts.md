# الخطوط

لا تُجمّع kou الخطوط معها. بدلاً من ذلك، وعلى غرار [ort](https://crates.io/crates/ort) لـ ONNX Runtime، تقوم بجلب عائلة خطوط منسّقة إلى ذاكرة تخزين مؤقت مشتركة (مرة واحدة، ثم تُخزَّن لجميع مرات التشغيل اللاحقة) وتحدد موقعها بشفافية. يقرن [`FontSet`] وجهاً رئيسياً أحادي المسافة مع وجه احتياطي اختياري لـ CJK؛ ويقوم المُنشئ بتحميلهما باستخدام `ab_glyph`، ثم لكل نقطة رمز، يختار أول وجه يغطيها — بحيث يمزج المنشئ الواحد بين اللاتينية وCJK دون فراغات.

## العائلات

| الخانة | قيمة `KOU_FONT_*` | العائلة |
|------|--------------------|--------|
| الرئيسي (افتراضي) | `fira-code` | Fira Code |
| الرئيسي | `jetbrains-mono` | JetBrains Mono |
| CJK (افتراضي) | `sarasa` | Sarasa Mono SC (更纱黑体) |
| CJK | `sourcehansans` | Source Han Sans SC (思源黑体) |
| CJK | `smileysans` | Smiley Sans (得意黑) |
| CJK | `none` | تعطيل الاحتياطي |

## ترتيب الحل

1. **ملف صريح** — `KOU_FONT_PATH` (الرئيسي) / `KOU_FONT_CJK_PATH`.
2. **الذاكرة المؤقتة المشتركة** — `<cache>/kou/fonts/<family>.ttf|otf`.
3. **التنزيل أثناء التشغيل** — تقوم ميزة `font-fetch` بجلب العائلة من مصدر عام (GitHub / jsDelivr) إلى الذاكرة المؤقتة.

## مفاتيح التحكم

| المتغير | الغرض |
|-----|---------|
| `KOU_FONT_PRIMARY` | تجاوز العائلة الرئيسية. |
| `KOU_FONT_CJK` | تجاوز / تعطيل عائلة CJK. |
| `KOU_FONT_MIRROR` | استبدال مضيف GitHub / jsDelivr الأساسي بنسخة مرآة (مثلاً للاستخدام خلف شبكة مقيدة). |
| `KOU_DOWNLOAD_PROXY` | توجيه تنزيلات الخطوط عبر وكيل `http://` / `https://` / `socks5://`. |
| `KOU_DOWNLOAD_TIMEOUT_SECS` | مهلة كل طلب (الافتراضي 120). |
| `KOU_SKIP_FONT_FETCH` | تعطيل الجلب أثناء التشغيل (استخدام الذاكرة المؤقتة / المسارات الصريحة فقط). |

## بدون خطوط على الإطلاق

إذا تعذر حل أي وجه، تكون `FontCache` فارغة ويتدهور المنشئ لرسم كتلة صلبة لكل خلية غير فارغة — بحيث لا تزال kou تُنتج *شيئاً* بدون خطوط، ولا تتعطل أبداً عند فقدان ملف.
